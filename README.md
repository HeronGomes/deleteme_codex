# deleteme_codex

## Visao geral
Repositorio minimo usado para experimentos com o Codex CLI e automacao de fluxos de trabalho em um ambiente seguro. O objetivo e permitir que agentes realizem operacoes de leitura e escrita sem comprometer codigo sensivel.

## Objetivos do projeto
- Demonstrar como o agente identifica e modifica arquivos.
- Servir como sandbox para validar comandos do Codex CLI.
- Manter um ponto de partida simples para criar exemplos e tutoriais.

## Estrutura do repositorio
